# Nostra – E-Commerce Website

Nostra is a responsive multi-page e-commerce website developed using **HTML, CSS, and JavaScript**.
The project demonstrates core frontend development concepts such as responsive layout design, DOM manipulation, and dynamic product search and filtering.

This project was created as part of a frontend development assignment to build a functional and user-friendly e-commerce interface.

---

## Project Objective

The goal of this project is to design and develop a **responsive e-commerce website** with multiple pages and interactive functionality using JavaScript.

---

## Pages Included

* **Home Page (`index.html`)**
  Displays the main landing section of the website with product highlights and navigation.

* **Collections Page (`collections.html`)**
  Displays available products and includes search and filter functionality.

* **Contact Us Page (`contact.html`)**
  Allows users to contact the business through a simple form or contact information section.

---

## Key Features

* Responsive design compatible with **desktop, tablet, and mobile devices**
* Multi-page website structure
* **Real-time product search** functionality
* **Product filtering** using JavaScript
* Clean and user-friendly interface
* Organized file structure for maintainability

---

## Technologies Used

* **HTML5** – Page structure and semantic layout
* **CSS3** – Styling and responsive design using Flexbox/Grid
* **JavaScript (ES6)** – Search functionality, filtering logic, and DOM manipulation

## Functional Requirements

### Search Functionality

Users can search for products by typing keywords in the search bar on the **Collections page**.
JavaScript dynamically filters and displays matching products in real time.

### Filter Functionality

Users can filter products based on categories or tags using JavaScript.
